# Navigation Bar

The navigation bar is a bottom-fixed bar with 3-5 evenly-spaced destinations, following the
[Material 3 spec](https://m3.material.io/components/navigation-bar/specs). Each destination stacks
a 24dp icon above a label, with an active-indicator pill behind the active destination's icon.
Mark the current destination with the `active` class on its `navigation-bar__item`.
